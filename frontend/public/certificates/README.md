# Certificate Images

Please upload your certificate images to this directory with the following names:

1. **cloud-computing.jpg** - NPTEL Cloud Computing certificate
2. **networking.jpg** - The Bits and Bytes of Computer Networking certificate

## Recommended Image Specifications:
- Format: JPG, PNG, or WebP
- Dimensions: Minimum 800x600 pixels
- File size: Less than 2MB for optimal loading

## File Naming:
- cloud-computing.jpg (or .png or .webp)
- networking.jpg (or .png or .webp)

The images will be automatically displayed in the certifications section of your portfolio.
