# Project Cover Images

Please upload your project cover images to this directory with the following names:

1. **fresh-meat-hub-cover.jpg** - Cover image for Fresh Meat Hub project
2. **metro-route-finder-cover.jpg** - Cover image for Metro Route Finder project

## Recommended Image Specifications:
- Format: JPG, PNG, or WebP
- Dimensions: Minimum 1200x600 pixels (2:1 aspect ratio recommended)
- File size: Less than 2MB for optimal loading

## File Naming:
- fresh-meat-hub-cover.jpg (or .png or .webp)
- metro-route-finder-cover.jpg (or .png or .webp)

The images will be automatically displayed in the projects section of your portfolio.
